
-- SUMMARY --

The Adyax Test module creates page (adyax-test-page), 20 test nodes, view (by the feature), menu item in the main menu.
All nodes have additional field "Adax info" in the node settings area (vertical tabs).

-- REQUIREMENTS --

Views, Features modules.

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/895232 for further information.
